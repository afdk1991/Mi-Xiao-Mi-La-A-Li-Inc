# 全平台小程序 - UniApp

使用UniApp框架开发，支持微信、支付宝、抖音、百度等多个平台小程序。

## 项目结构

```
uni-app-mini/
├── src/
│   ├── pages/
│   ├── components/
│   ├── static/
│   └── main.js
├── manifest.json
├── pages.json
└── package.json
```

## 支持平台

- 微信小程序
- 支付宝小程序
- 抖音小程序
- 百度小程序
- H5
- APP (可选)

## 功能模块

- 首页
- 商品分类
- 商品详情
- 购物车
- 订单管理
- 个人中心

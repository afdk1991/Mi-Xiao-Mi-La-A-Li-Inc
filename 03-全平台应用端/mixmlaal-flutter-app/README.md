# Flutter APP - Android / iOS 双端应用

使用Flutter开发的跨平台移动应用，一套代码同时支持Android和iOS。

## 项目结构

```
flutter-app/
├── lib/
│   ├── main.dart
│   ├── pages/
│   ├── components/
│   ├── models/
│   └── services/
├── android/
├── ios/
└── pubspec.yaml
```

## 技术栈

- Flutter 3.x
- Dart 3.x
- Provider / Riverpod (状态管理)
- Dio (网络请求)
- GetX (可选)

## 功能模块

- 用户登录/注册
- 商品浏览
- 购物车
- 订单管理
- 个人中心
- 支付集成

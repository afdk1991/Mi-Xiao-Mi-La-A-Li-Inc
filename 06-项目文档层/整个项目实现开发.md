# 亿级商用超级平台 · 整体实现开发文档

---

## 一、项目概述

### 1.1 项目定位
亿级商用全平台超级综合体，覆盖Web网站、全平台小程序、安卓/iOS APP、Windows/Mac桌面软件、通用后端接口、总后台管理系统。

### 1.2 技术栈选型
| 层级 | 技术方案 |
|------|----------|
| 前端 | Vue3 + Vite + Element Plus, React18 + Ant Design, HTML5 + CSS3 + JS, UniApp, Flutter, Electron |
| 后端 | Java Spring Cloud, Python FastAPI, Node NestJS |
| 数据库 | MySQL 8.0 (10库分表), Redis |
| 部署 | Docker Compose, Nginx |

### 1.3 业务能力
电商商城、博客资讯、考勤人事、文件私有云、数据可视化、本地生活、社交IM、会员付费、AI工具、招商入驻、财务核销等全行业全核心功能。

## 三、前端实现（01-前端集群）

### 3.1 Vue3 PC用户前台 (web-vue-pc)
**技术栈**: Vue 3.4 + Vite 5.1 + Element Plus 2.6 + Vue Router + Pinia

**页面清单**:
| 页面 | 路由 | 功能 |
|------|------|------|
| 首页 | / | 轮播 Banner、商品分类、热门商品、服务承诺 |
| 登录 | /login | 用户登录、注册链接 |
| 注册 | /register | 用户注册 |
| 商品列表 | /goods | 商品搜索、分类筛选、分页 |
| 商品详情 | /goods/:id | 商品信息、规格选择、加入购物车 |
| 购物车 | /cart | 购物车管理、数量修改、结算 |
| 订单管理 | /order | 订单列表、订单详情、支付、取消 |
| 用户中心 | /user | 个人资料、收货地址、钱包、积分 |

**核心组件**:
- `HomeBanner.vue` - 首页轮播组件
- `GoodsCategory.vue` - 商品分类组件
- `Service承诺.vue` - 服务承诺组件
- `SearchBar.vue` - 搜索框组件
- `CategorySidebar.vue` - 分类侧边栏
- `Header.vue` - 全局顶部导航
- `Footer.vue` - 全局底部

### 3.2 React管理后台 (web-react-admin)
**技术栈**: React 18.2 + Vite 5.1 + Ant Design 5.15 + React Router

**页面清单**:
| 页面 | 路由 | 功能 |
|------|------|------|
| 登录 | /login | 管理员登录 |
| Dashboard | /dashboard | 数据概览、统计图表 |
| 用户管理 | /user | 用户列表、搜索筛选、状态管理 |
| 商品管理 | /goods | 商品CRUD、分类管理、图片上传 |
| 订单管理 | /order | 订单列表、状态筛选、详情查看 |

### 3.3 其他前端
- `web-html-static` - 原生HTML5静态官网
- `web-h5-mobile` - Vue3 + Vant移动端H5

## 四、后端实现（02-后端微服务集群）

### 4.1 Java Spring Cloud微服务
**架构**: Spring Boot 3.2 + Spring Cloud Alibaba + Nacos + OpenFeign

**模块划分**:
| 模块 | 端口 | 功能 |
|------|------|------|
| cloud-common | - | 公共模块（统一返回、异常处理、工具类） |
| cloud-gateway | 8888 | API统一入口、路由转发、限流 |
| cloud-system | 9001 | 系统权限、角色菜单管理员 |
| cloud-user | 9002 | 用户服务、会员管理 |
| cloud-shop | 9003 | 商品服务、分类管理 |
| cloud-order | 9004 | 订单服务、支付处理 |

**核心业务类**:
- `SysAdminService.java` - 管理员服务接口
- `SysAdminServiceImpl.java` - 管理员服务实现
- `CloudUserController.java` - 用户控制器
- `CloudOrderController.java` - 订单控制器

### 4.2 Python FastAPI服务
**技术栈**: FastAPI + Uvicorn + Pydantic + SQLAlchemy

**接口模块** (70+接口):
| 模块 | 接口数 | 功能 |
|------|--------|------|
| user.py | 11 | 用户资料、收货地址、积分钱包、绑定手机邮箱 |
| order.py | 10 | 订单CRUD、支付、退款、物流、统计 |
| data.py | 7 | 数据概览、趋势、实时统计、用户/商品/订单/营收统计 |
| ai.py | 6 | 智能推荐、相似商品、热门搜索、流行趋势 |
| file.py | 8 | 文件上传、列表、文件夹、分享、下载 |
| article.py | 8 | 文章CRUD、分类、点赞、评论 |
| attend.py | 9 | 考勤打卡、请假加班、统计 |
| im.py | 8 | 联系人、消息、群聊、未读消息 |

**启动命令**:
```bash
cd 02-后端微服务集群/python-fastapi
python -m venv venv && venv\Scripts\activate
pip install -r requirements.txt
python main.py
# 访问 http://localhost:8001/docs 查看API文档
```

### 4.3 Node NestJS网关+IM
**技术栈**: NestJS + TypeScript + WebSocket + JWT

**核心模块**:
- API网关（路由代理、鉴权）
- IM即时通讯（WebSocket实时消息）
- 用户服务

**端口**:
- HTTP网关: 8002
- WebSocket IM: 8003

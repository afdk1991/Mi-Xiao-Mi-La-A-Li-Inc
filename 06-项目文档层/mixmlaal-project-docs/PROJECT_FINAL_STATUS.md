## 技术实现

### 后端技术栈
- **运行环境**: Node.js v24.6.0
- **Web框架**: Express.js
- **数据库**: MongoDB v4.4+
- **ODM**: Mongoose
- **认证**: JWT + bcryptjs
- **安全**: Helmet, CORS, 速率限制
- **日志**: Morgan
- **压缩**: Compression

### 前端技术栈
- **HTML5**: 语义化标签
- **CSS框架**: Tailwind CSS
- **图表库**: Chart.js
- **响应式设计**: 移动端适配

### 部署技术
- **容器化**: Docker
- **环境管理**: dotenv
- **进程管理**: PM2 (可选)

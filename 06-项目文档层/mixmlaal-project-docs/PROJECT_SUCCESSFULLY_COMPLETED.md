## 📖 使用说明

### 启动项目
```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问地址
http://localhost:3002
```

### Docker部署
```bash
# 构建镜像
docker build -t enterprise-project-management-system .

# 运行容器
docker run -p 3002:3002 enterprise-project-management-system
```

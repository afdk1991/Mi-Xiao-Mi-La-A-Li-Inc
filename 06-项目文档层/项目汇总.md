## 🚀 快速启动

### 方式一：使用启动脚本（最简单）
```bash
# 双击运行
start.bat
```

### 方式二：手动启动前端
```bash
# Vue3用户前台
cd 01-前端集群/web-vue-pc
npm install && npm run dev

# React管理后台
cd 01-前端集群/web-react-admin
npm install && npm run dev
```

## 📈 项目亮点

### 1. 全平台覆盖
- Web端：Vue3 + React
- 移动端：H5 + UniApp + Flutter
- 桌面端：Electron
- 全平台一套API，高效复用

### 2. 微服务架构
- 10库分表，业务垂直拆分
- 负载均衡，高可用设计
- 服务注册与发现

### 3. 完整功能
- 电商核心：商品、订单、购物车
- 管理后台：数据统计、用户/商品/订单管理
- 数据服务：可视化统计、AI推荐
- IM通讯：WebSocket实时聊天

### 4. 商用级标准
- 统一返回格式
- 统一异常处理
- Redis缓存
- Docker容器化部署
- Nginx反向代理

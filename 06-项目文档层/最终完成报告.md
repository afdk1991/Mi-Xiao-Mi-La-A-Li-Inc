## 🚀 快速启动

### 方式一：使用启动脚本
```bash
双击运行 start.bat
```

### 方式二：手动启动
```bash
# Vue3 PC前台
cd 01-前端集群/web-vue-pc && npm install && npm run dev

# React管理后台
cd 01-前端集群/web-react-admin && npm install && npm run dev

# Python FastAPI
cd 02-后端微服务集群/python-fastapi
python -m venv venv && venv\Scripts\activate
pip install -r requirements.txt
python main.py

# Node NestJS
cd 02-后端微服务集群/node-nest-gateway && npm install && npm run start:dev
```

### 方式三：Docker部署
```bash
cd 05-部署运维层/docker-compose
docker-compose -f docker-compose-full.yml up -d
```

## 七、全站通用：密钥安全 \+ 防盗刷 \+ 额度管控（必看）

### 1）防盗刷硬性安全规范（全部平台通用）

① API Key、Token、AccessKey **绝对不能上传GitHub、码云、前端公开代码**；② 所有密钥必须绑定：域名白名单 / 小程序AppID / APP包名 / 可信IP；③ 不用的应用直接删除密钥、下线应用；④ 定期轮换密钥，降低泄露风险；⑤ 只开放项目必需接口，关闭全部闲置API权限。

### 2）额度预警怎么开（防止突然停机）

国内四家（高德/百度/腾讯/阿里云）：控制台费用中心 → 额度告警 → 设置**80%额度触发短信/邮箱提醒**。海外两家（谷歌/Mapbox）：后台设置月度消费预算封顶，自动拦截超额扣费。开源小众平台：无官方预警，必须前端\+后端双层缓存控量。

### 3）低成本控费核心技巧（个人必学）

高频固定地址、常用点位、静态路线全部做本地缓存；重复请求不要反复调用API；测试环境单独用测试Key，不占用生产额度；个人账号一律关闭自动续费，额度用尽自动停机不扣费；多场景组合混用，不单一依赖一家平台。
